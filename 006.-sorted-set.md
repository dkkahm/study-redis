### commands
  - zadd key score1 member1 score2 member2 ...
    - client.zadd([key, score1, member1, score2, member2, ...])
  - zrange key start stop [withscores]
    - start and stop are rank
  - zrevrange key start stop [withscores]
  - zrem key member1 member2
  - zscore key member
  - zrank key member
    - rank of member
    - lower to higher
    - starts from 0
  - zrevrank key member
