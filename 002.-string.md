### commands
  - set key value
  - get key
  - keys p*
  - mset k1 v1 k2 v2 ...
  - mget k1 k2 ...
  - expire key seconds
  - ttl key
    - -1: key exists, expires is not set
    - -2: expires or key is not exists
    - other: remain seconds
  - incr key
    - increase by 1
  - decr key
    - decrease by 1
  - incrby key step
    - increase by step
    - step can be minus
  - desrby key step
    - decrease by step
    - step can be minus
  - incrbyfloat key float_step
